# Flask Greeting App

A simple Flask application that greets the user.

## How to Run

1. Install Flask:
   ```bash
   pip install Flask
   ```

2. Run the app:
   ```bash
   flask run
   ```

3. Open in your browser:
   ```
   http://adress:port/
   ```

4. Customize the greeting:
   ```
   http://adress:port/?name=YourName
   ```

## Files

- `app.py` - Main Flask file.
- `templates/index.html` - HTML template.
